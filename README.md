## 🚀 Machine Learning & Data Science Projects

### 💳 Credit Risk Prediction
Developing a machine learning model using the Lending Club dataset to predict borrower default risk. This project involves data cleaning, feature engineering, exploratory data analysis (EDA), and training classification models with Python and Scikit-learn.

### 🩺 Gestational Diabetes Risk Prediction
Building binary classification models using Logistic Regression and Random Forest to predict gestational diabetes. Evaluating performance with ROC-AUC, Precision, Recall, and F1-score while addressing the challenges of limited clinical data.

### 💧 Water Consumption Analytics
Developed a predictive analytics system for monitoring water consumption using Python, Streamlit, SQL, and interactive dashboards. This project was completed during my internship and published in the *International Journal of Science, Engineering and Technology (IJSET)*.
